# Skillforge-
 This is AI Driven Adaptive Learning and Exam Generator project
 SkillForge is a next-generation educational platform designed to dismantle the "one-size-fits-all" approach to learning. By leveraging Artificial Intelligence, the application creates a dynamic ecosystem where course content adapts in real-time to a learner's proficiency, and assessments are automatically generated to ensure academic integrity. It bridges the gap between personalized tutoring and scalable classroom management, serving students, educators, and corporate trainers alike.
